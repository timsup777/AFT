# Affûteur à Vélo

Site web professionnel pour service d'affûtage à domicile à vélo - Paris

## Description

Site vitrine pour un artisan affûteur rémouleur proposant un service unique d'affûtage à domicile à vélo dans tout Paris.

## Fonctionnalités

- 🏠 Page d'accueil avec slider
- 🔪 Présentation des lames et outils affûtés
- 💰 Grille tarifaire complète
- 👤 Section À propos
- ⭐ Avis clients
- ❓ FAQ
- 📅 Système de réservation en ligne (Calendly)
- 📱 Design responsive pour mobile

## Technologies

- HTML5
- CSS3
- JavaScript vanilla
- Calendly (système de réservation)
