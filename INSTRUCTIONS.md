# Instructions pour ajouter la photo du vélo

## Étape 1 : Sauvegarder la photo

1. Sauvegardez la photo du vélo-cargo que vous avez envoyée
2. Renommez-la en : `velo-affutage.jpg`
3. Placez-la dans le dossier : `images/` (qui a été créé automatiquement)

Chemin complet : `/Users/timotheeberthelot/Desktop/affuteurs a velo/images/velo-affutage.jpg`

## Étape 2 : Vérifier

Une fois la photo placée, rafraîchissez le site sur http://localhost:8000

La photo du vélo apparaîtra en arrière-plan du slider, exactement comme sur le site Elian Affuteur.

## Résultat attendu

- La photo du vélo sera visible en arrière-plan
- Le texte blanc restera lisible grâce à l'overlay sombre
- Le slider changera automatiquement toutes les 5 secondes
- Les animations et le design resteront identiques au site de référence

## Ajouter d'autres photos (optionnel)

Si vous voulez différentes photos pour chaque slide :

1. Ajoutez les photos dans le dossier `images/`
2. Nommez-les : `slide1.jpg`, `slide2.jpg`, `slide3.jpg`
3. Modifiez le fichier `styles.css` lignes 190-200 pour changer les chemins
